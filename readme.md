## py-fingerprint

### Installation

-   [Check Python](https://docs.python.org/)
-   [Check NodeJs](https://nodejs.org/en)


### Usage
```sh
pip install -U Pyzk
```

#### Exporting Attendance Data from fingerprint
```sh
.\export.py
```
OR
```sh
py .\export.py
```
OR 
```sh
python .\export.py
```

#### Check Attendance Data
```sh
npm install
```
Then
```sh
npm run dev
```
